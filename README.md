# Lista 1 - Lógica de Programação

Exercícios desenvolvidos em Python e Java.

## Conteúdo

### Parte 1 — Fundamentos de Programação

* Saudação
* Soma de dois números
* Média de três números
* Tabuada
* Par ou Ímpar
* Contador de vogais
* Maior, menor e média
* Jogo de adivinhação
* Calculadora com funções
* Manipulação de arquivo TXT (lista de tarefas)

### Parte 2 — Monitoramento de Recursos do Sistema

Programas desenvolvidos em Python para monitorar recursos do sistema operacional.

* Monitor simples de memória RAM
* Monitor de RAM com alerta de uso
* Gerenciador de espaço em disco
* Alerta de pouco espaço em disco
* Monitor de tráfego de rede (download e upload)
* Monitor de desempenho da CPU
* Histórico de uso da CPU (logger)
* Informações detalhadas do processador
* Listagem de dispositivos de entrada e saída
* Painel integrado de monitoramento (RAM + CPU + Disco + Rede)

## Linguagens utilizadas

* Python
* Java

## Bibliotecas utilizadas (Python)

Alguns exercícios utilizam bibliotecas externas para acessar informações do sistema.

Principais bibliotecas:

* psutil → acesso a informações de CPU, RAM, disco e rede
* py-cpuinfo → informações detalhadas do processador

Instalação das dependências:

pip install -r requirements.txt

ou

python -m pip install -r requirements.txt

## Execução dos programas

Para executar qualquer exercício em Python:

python nome_do_arquivo.py

Exemplo:

python monitor_ram.py
